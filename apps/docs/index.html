Getting Started

Install

pnpm add @blazon/core @blazon/country-poland @blazon/types

Tree-shakeable static imports

// Import only the cities you need — the rest are tree-shaken away
import { warszawa, krakow, wroclaw } from '@blazon/country-poland';

console.log(warszawa.name);  // 'Herb Warszawy'
console.log(krakow.svg);     // inline SVG string

Or register a lazy HTTP loader

import {
  registerCountry,
  getCountryRegistry,
  getById,
  searchRegistry,
} from '@blazon/core';

// Register a lazy loader for Poland
registerCountry('PL', () =>
  fetch('/registries/pl.json').then(r => r.json())
);

// Load on demand
const registry = await getCountryRegistry('PL');
console.log(registry?.entries.length, 'entries loaded');

// Find by ID or search
const city = getById('pl-city-warszawa');
const cities = searchRegistry({ countryCode: 'PL', level: 'city', limit: 10 });

Install

pnpm add @blazon/ngx @blazon/core @blazon/country-poland @blazon/types

Tree-shakeable — provide only what you need

// app.config.ts
import { warszawa, krakow } from '@blazon/country-poland';
import { provideBlazonIcons } from '@blazon/ngx';

export const appConfig: ApplicationConfig = {
  providers: [
    // Only Warszawa and Kraków are bundled — everything else is tree-shaken away
    provideBlazonIcons([warszawa, krakow]),
  ],
};

Or load the full collection lazily

// app.config.ts
import { provideBlazonIcons } from '@blazon/ngx';

export const appConfig: ApplicationConfig = {
  providers: [
    provideBlazonIcons({
      loaders: {
        PL: () => fetch('/registries/pl.json').then(r => r.json()),
      },
      preload: ['PL'],
    }),
  ],
};

Use the component

// my.component.ts
import { BlazonIcon } from '@blazon/ngx';

@Component({
  imports: [BlazonIcon],
  template: `
    <blazon-icon id="pl-city-warszawa" [size]="64" />
    <blazon-icon id="pl-city-krakow"   [size]="64" />
  `,
})
export class MyComponent {}

Core API

getById(id)

Retrieves a coat of arms by unique ID from loaded registries. Returns CoatOfArms | undefined.

searchRegistry(query)

Searches all loaded registries. Supports text, country, level, type, tags, and pagination filters.

getCountryRegistry(code)

Lazily loads and returns the registry for a country. Returns Promise<CountryRegistry | undefined>.

registerCountry(code, loader)

Registers a lazy loader for a country's data. The loader is invoked at most once; results are cached.

Packages

@blazon/types

Zero-runtime TypeScript type definitions. All domain types: CoatOfArms, CountryRegistry, CoatMetadata, AssetLicense, SearchQuery.

@blazon/core

Framework-agnostic registry engine. Singleton registry, search, validation, and lazy HTTP loading. Pure engine — no country data included.

@blazon/country-poland

495 Polish city coats of arms as tree-shakeable named exports. import { warszawa, krakow } from '@blazon/country-poland'

@blazon/ngx

Angular adapter (v16+). Use provideBlazonIcons([warszawa]) for static tree-shaking or a lazy-loader config for HTTP loading.
